# Prefer curated Location Resolutions

Site Owner-published Location Resolutions are authoritative over browser persistence and external geocoding, including when retained as a last validated stale snapshot during relay outages. Nominatim and browser data only discover missing coordinates, failed lookups expire after 24 hours, broader matches remain visibly approximate, and only exact discoveries are automatically promoted by the Site Owner; this preserves corrections and provenance behind one location-resolution seam.
